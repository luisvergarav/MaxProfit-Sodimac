# MaxProfit-Sodimac
 JavaScript code that returns the maximum profit that could have been made by buying a stock at a price and then selling 
 the stock later on (buy once, sell once). 
 Examples
 
 A. Input: [44, 30, 22, 32, 35, 30, 41, 38, 15]
   Output: 19

B. Input: [2,3,4,2]
   Output: 2

C. Input: [50, 30, 4, 2]
   Output: -1
